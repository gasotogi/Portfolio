Sobre


Sou Técnica em informática pelo Instituto Federal do Paraná, graduanda em Engenharia Eletrônica pela Universidade Tecnológica Federal do Paraná, e por isso me considero extremamente apaixonada e dedicada à tecnologia. Possuo alguns cursos de especialização em UX/UI design e desenvolvimento web voltado à experiência do usuário.


Prezo muito pelo desenvolvimento de componentes reutilizáveis, design Systems, animações fluidas e por evoluir continuamente a performance e a acessibilidade das minhas aplicações.

Algumas das linguagens e frameworks quais sou entusiasta são:

  • JavaScript

    JavaScript é uma linguagem de programação versátil, amplamente usada para desenvolvimento web, tanto no frontend quanto no backend.

  • Python

    Python é conhecida por sua simplicidade e legibilidade, sendo amplamente utilizada em ciência de dados, automação e inteligência artificial.

  • C++

    C++ é uma linguagem poderosa usada no desenvolvimento de sistemas, jogos e aplicações de alto desempenho.

  • React.js

    React.js é uma biblioteca JavaScript para construir interfaces de usuário de forma declarativa e eficiente, usando componentes reutilizáveis.

  • Quasar

    Quasar é um framework baseado em Vue.js para construir aplicações SPA, SSR, PWA, mobile e desktop com alta performance e design consistente.

  • Vue.js

    Vue.js é um framework progressivo para construção de interfaces interativas e reativas, com foco na simplicidade e flexibilidade.

  • CSS

    CSS é a linguagem de estilo usada para descrever a apresentação de páginas web, controlando cores, layouts, fontes e responsividade.
